### my-elasticsearch-autocomplete-spring-boot-react-workshop

```
Bu projede Backend tarafında Spring Boot ile Elasticsearch NoSQL veritabanını kullandım.
Ön yüzde ise REACT'ı kullanarak Autocomplete projesi yazdım.
Amaç ürünlerin hızlıca aranabilmesi ve kullanıcıya tavsiyelerde bulunabilmesidir.
Leb demeden leblebiyi tavsiye ederse hoş olur.
Arama işlemleri için UI(Frontend) kısmı yazıldı. Arayüzdeki Input'a değer girildikçe aşağı doğru ürünleri tamamlamakta.
```
